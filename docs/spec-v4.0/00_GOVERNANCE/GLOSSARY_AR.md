# قاموس المصطلحات

| المصطلح | التعريف الملزم |
|---|---|
| المكتب | الجهة المسجلة التي تنشر الرحلات أو تبيع الحجوزات وتدير التشغيل المحلي. |
| الناقل | شركة النقل الفعلية المسؤولة عن المركبة والسائق وتنفيذ الرحلة. قد تكون المكتب نفسه أو جهة مرتبطة. |
| الفرع | نقطة تشغيل تابعة لمكتب، لها موظفون وصندوق وتقارير ونطاق صلاحيات. |
| الرحلة | تشغيل محدد لمسار في تاريخ ووقت وبولمان وسعر وسياسة نافذة. |
| المسار | علاقة تشغيلية بين نقاط صعود ونزول مرتبة، وليست مجرد مدينتين. |
| PNR | مرجع حجز قصير موجه للمستخدم، غير قابل للتخمين بسهولة ولا يساوي المفتاح الداخلي. |
| Seat Hold | حجز مؤقت للمقاعد ينتهي تلقائيًا إن لم يكتمل الحجز. |
| Snapshot | نسخة ثابتة من السعر والسياسة والمهلة والعمولة وقت موافقة الزبون. |
| Manifest | قائمة ركاب الرحلة المعتمدة للتشغيل والصعود. |
| No-show | راكب لم يصعد بعد انقضاء شروط الحضور والإغلاق وعدم وجود خطأ من المكتب. |
| Denied Boarding | منع راكب صالح من الصعود، ويُفتح له مسار دعم وتحقيق مستقل. |
| Ledger | دفتر أحداث مالي مزدوج القيد لا تُعدل قيوده بعد ترحيلها. |
| Settlement | تسوية صافي الأموال بين المنصة والمكتب بعد العمولات والاستردادات والتجميد. |
| Tenant | نطاق مكتب معزول بياناتيًا وصلاحيًا عن المكاتب الأخرى. |
| Feature Flag | مفتاح تشغيل تدريجي لميزة دون نشر نسخة جديدة، مع سجل وتاريخ نفاذ. |
