# المصادقة والجلسات وMFA

## الحسابات

- حساب موحد يمكن أن يملك عضوية عميل أو مكتب أو منصة، لكن الصلاحيات مستقلة.
- البريد/الهاتف يُتحقق منه قبل العمليات الحساسة.
- كلمات المرور تُخزن بهاشر Django الآمن وتخضع لسياسة منع كلمات المرور الشائعة.

## الجلسات

- معرف جلسة عشوائي ودوران بعد الدخول وتغيير الصلاحية.
- عرض الأجهزة والجلسات وإلغاؤها.
- انتهاء خمول ومدة قصوى مختلفة للحسابات الحساسة.
- إعادة تحقق للعمليات: تغيير حساب التسوية، تصدير بيانات واسعة، تعديل MFA، استرداد مرتفع.

## MFA

إلزامي لصاحب المكتب والأدوار المالية والإدارية والمنصة. يفضل TOTP/WebAuthn عند توفره، وتستخدم رموز استرداد محمية. SMS ليس العامل الوحيد للأدوار الحرجة عند توفر بديل.

## الحماية من الاستيلاء

- تنبيه جهاز جديد.
- Rate limits حسب الحساب والجهاز والشبكة دون حظر نهائي مبني على IP فقط.
- قفل مؤقت وتحديات إضافية عند المخاطر.
- إبطال الجلسات بعد تغيير كلمة المرور أو اكتشاف اختراق.
