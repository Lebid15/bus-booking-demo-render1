# الاستجابة للحوادث الأمنية

## المراحل

1. اكتشاف وتصنيف.
2. تعيين قائد حادث وكاتب Timeline.
3. احتواء دون تدمير الأدلة.
4. تحليل النطاق والبيانات المتأثرة.
5. استئصال السبب وتغيير الأسرار/الجلسات.
6. استعادة ومراقبة مشددة.
7. إخطار قانوني ومستخدمين عند الالتزام.
8. Postmortem بلا لوم وخطة تصحيح بمالكين ومواعيد.

## قواعد

- لا تستخدم قناة مخترقة للتنسيق.
- لا تعطل Audit لإخفاء الضوضاء.
- تتغير مفاتيح مزود الدفع/الإشعار وفق Runbook.
- يتم حفظ صور الأنظمة وLogs والأدلة بسلسلة حيازة.
- لا تعلن أرقامًا أو أسبابًا قبل التحقق.
