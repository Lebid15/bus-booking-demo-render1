# أمن رفع الملفات

1. ينشئ الخادم Upload Intent بمستخدم وغرض وحجم وأنواع مسموحة.
2. يرفع الملف إلى Bucket خاص بمفتاح عشوائي، لا باسم المستخدم.
3. يتحقق من MIME الفعلي والامتداد والحجم والتوقيع السحري.
4. يمر عبر فحص برمجيات خبيثة؛ تبقى الحالة Quarantined حتى النجاح.
5. تُستخرج Metadata الضرورية فقط، وتزال Metadata غير اللازمة عند إمكانها.
6. روابط العرض موقعة وقصيرة العمر، ولا تجعل Bucket عامًا.
7. يمنع SVG/HTML أو الملفات التنفيذية في السياقات الحساسة ما لم تعقم.
8. يسجل الوصول للمستندات المقيدة.
9. تطبق سياسة احتفاظ وحذف منفصلة للمستندات المرفوضة والمنتهية.
