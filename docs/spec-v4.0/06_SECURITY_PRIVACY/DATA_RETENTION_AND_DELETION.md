# الاحتفاظ والحذف

| الفئة | القاعدة الأولية | نهاية المدة |
|---|---|---|
| Seat Holds غير المكتملة | قصيرة للتشغيل والتحقيق | حذف/تجميع مجهول |
| حجوزات وتذاكر | حسب الالتزام المالي والقانوني | إخفاء الهوية حيث أمكن |
| Ledger/Settlement | مدة مالية ملزمة | لا حذف قبل المدة؛ أرشفة |
| مستندات تحقق مكتب | حتى انتهاء العلاقة + مدة قانونية | حذف آمن |
| إثباتات تحويل | حسب النزاع والمحاسبة | حذف/أرشفة محمية |
| دعم | حسب النوع وخطر النزاع | إخفاء بيانات زائدة |
| Audit | مدة أمنية/قانونية | أرشفة غير قابلة للتعديل |
| Analytics | أقل مدة وبمعرفات مستعارة | تجميع/حذف |

## إجراءات الحذف

- Job مجدول وقابل للتدقيق.
- Legal Hold يوقف الحذف لفئة/كيان محدد.
- حذف من النسخ الاحتياطية يتم بانتهاء دورة النسخ، مع منع الاستعادة الدائمة دون تطبيق حذف منطقي لاحق.
- طلب حذف الحساب لا يحذف حقوق مكتب/زبون آخر أو قيودًا مالية ملزمة.
