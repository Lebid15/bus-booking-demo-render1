# اختبار تجربة الاستخدام والوصول

## الأجهزة

360px Android شائع، هاتف متوسط، لوحي، سطح مكتب. اختبارات شبكات بطيئة وانقطاع متقطع.

## مهام

- حجز ضيف لمرافقين.
- فهم الفرق بين “مؤكد” و“مدفوع”.
- استرجاع حجز بـPNR.
- إلغاء ومعرفة المبلغ.
- موظف مكتب ينشئ حجزًا ويؤكد تحويلًا.
- موظف صعود يعالج QR غير صالح.

## الوصول

Keyboard-only، screen reader بالعربية، zoom 200%، contrast، reduced motion، رسائل الحقول، خريطة المقاعد، PDF/تذكرة قابلة للقراءة.
