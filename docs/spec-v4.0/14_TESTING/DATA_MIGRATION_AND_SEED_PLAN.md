# خطة البيانات الأولية والترحيل

## Seed مرجعي

الأدوار الأساسية، الصلاحيات، حالات النظام، قوالب السياسات، العملات المسموحة، Feature Flags، أنواع المستندات، وأكواد الأخطاء. لا تشمل أسعارًا أو مكاتب وهمية في الإنتاج.

## إدخال مكتب قائم

- تنظيف الأسماء والهواتف.
- إنشاء البولمان والمخططات.
- إدخال الرحلات المستقبلية فقط أو تاريخ محدود متفق عليه.
- مطابقة المقاعد الحالية قبل فتح البيع العام.
- تقرير فروقات وتوقيع المكتب.

## تحقق

counts، checksums، orphan checks، duplicate seats، financial opening balances، وعينة يدوية. خطة rollback قبل فتح الحجوزات.
