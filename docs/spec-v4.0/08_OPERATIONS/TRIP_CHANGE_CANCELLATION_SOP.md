# SOP تغيير أو إلغاء رحلة

## تغيير بسيط

سجل السبب، عدل، أخطر. لا يحتاج قبولًا إذا لم يغير الحق الجوهري وفق السياسة.

## تغيير جوهري

- أنشئ Change Proposal لا تعديلًا مباشرًا.
- احسب المتأثرين والبدائل والتكلفة.
- أرسل إشعارًا يحتاج قرارًا.
- لا تعتبر الصمت قبولًا.
- احفظ قرار كل حجز.

## إلغاء

- اختر السبب والمسؤولية.
- امنع حجوزات جديدة.
- أنشئ خيارات استرداد كامل/بديل.
- اعكس العمولة والتسويات.
- افتح حادثًا إذا كان واسعًا أو متكررًا.
