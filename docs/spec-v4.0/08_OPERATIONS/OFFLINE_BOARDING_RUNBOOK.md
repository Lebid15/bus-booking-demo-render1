# Runbook الصعود عند انقطاع الإنترنت

## قبل الانقطاع/الرحلة

- تنزيل Manifest مشفر وموقع ومحدود المدة لجهاز مسجل.
- يشمل أقل بيانات لازمة ونسخة trip revision.
- اختبار فتحه دون اتصال.

## أثناء الانقطاع

- يسمح بالصعود والتحقق فقط.
- لا حجوزات جديدة ولا تغيير راكب ولا استرداد.
- يسجل الجهاز events بمعرفات فريدة ووقت محلي موثوق قدر الإمكان.

## بعد العودة

- مزامنة Idempotent.
- كشف QR صعد Online وOffline أو اختلاف نسخة الرحلة.
- لا يحذف الحدث المتعارض؛ يفتح مراجعة.
- يمسح Manifest من الجهاز بعد المدة/المزامنة وفق السياسة.
