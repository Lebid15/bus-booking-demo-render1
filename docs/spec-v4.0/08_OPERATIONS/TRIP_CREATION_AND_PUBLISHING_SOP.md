# SOP إنشاء ونشر الرحلة

1. اختيار المكتب/الفرع والناقل والمسار.
2. تحديد وقت الانطلاق والوصول المتوقع ونقاط الصعود/النزول.
3. تعيين بولمان بمخطط منشور وسائق بوثائق نافذة.
4. تحديد السعر والعملة والخدمات والأمتعة.
5. اختيار طرق الدفع ومهلة الدفع وسياسة الإلغاء والصعود.
6. تشغيل Readiness Check: مستندات، تعارض مركبة/سائق، مخطط، سعر، سياسة، اتصال مكتب.
7. حفظ مسودة ثم نشر.
8. بعد أول حجز، التعديلات الجوهرية تمر عبر Workflow ولا تعدل السجل بصمت.

## منع التعارض

لا يجوز تعيين المركبة أو السائق لرحلتين متداخلتين دون استثناء موثق يسمح بالزمن والموقع.
