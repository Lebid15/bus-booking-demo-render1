# تعليق المكتب والخروج المنظم

## الحالات

Active، Restricted، No New Bookings، Wind-down، Terminated، Archived.

## عند التعليق

- أوقف النشر والحجوزات الجديدة حسب الخطر.
- راجع كل رحلة قائمة: تنفيذ، نقل بموافقة، أو إلغاء.
- أبق وصولًا محدودًا للدعم والتسوية والتزامات العملاء.
- جمد المبالغ المرتبطة فقط.
- عين مسؤول متابعة ومواعيد.

## الخروج الطوعي

إشعار مسبق، إيقاف إنشاء رحلات بعد تاريخ، تسوية الرصيد، حفظ السجلات، إلغاء حسابات الموظفين، وإتاحة تصدير مشروع للبيانات المسموحة.
