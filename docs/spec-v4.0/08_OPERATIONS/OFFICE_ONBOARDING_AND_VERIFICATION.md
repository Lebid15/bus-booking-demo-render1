# تشغيل انضمام المكتب والتحقق

## المراحل

1. طلب أولي وبيانات اتصال.
2. تعريف نوع الجهة: ناقل، فرع شركة، مكتب مفوض.
3. مستندات التسجيل والترخيص والعنوان والممثل والمالك الفعلي وتفويض الناقل.
4. تحقق الحساب المالي بقناة مستقلة.
5. مراجعة تضارب/تكرار/مخاطر.
6. عقد وشروط وملف عمولة واشتراك.
7. قبول مشروط بحدود أو تفعيل كامل.
8. تدريب وتشغيل تجريبي.

## التجديد

تنبيهات 60/30/15/7/0 يوم. انتهاء مستند حرج يمنع تعيين جديد أو نشر رحلة، ولا يلغي التزامات قائمة بصمت.

## دليل القبول

نسخ مستندات، قرارات مراجعة، سجل الموافقات، اختبار فرع وموظف، رحلة تجريبية، وتأكيد حساب التسوية.
