# قائمة التشغيل اليومية للمكتب

## بداية اليوم

- مراجعة رحلات اليوم والوثائق والتنبيهات.
- التأكد من البولمان والسائق ونقاط الانطلاق.
- مراجعة الحجوزات غير المدفوعة والتحويلات.
- اختبار جهاز الصعود والطابعة/الهاتف.

## قبل كل رحلة

- تأكيد الموعد والبولمان.
- تنزيل Manifest طوارئ.
- معالجة تغييرات الركاب والمقاعد.
- التواصل مع المتأثرين بأي تغيير.

## نهاية اليوم

- إغلاق الرحلات المكتملة.
- مطابقة النقد والتحويلات والمدفوعات.
- مراجعة No-show وDenied Boarding.
- فتح فروقات بدل تعديلها.
- تسليم الحالات المفتوحة للوردية التالية رسميًا.
