# خطة الإطلاق التجريبي

## النطاق

ممر واحد مثل الرقة ↔ دمشق، 1–3 مكاتب متحقق منها، عدد محدود من الرحلات، معاملات حقيقية، وقنوات دفع مناسبة للمكتب.

## مراحل Pilot

1. Dry run داخلي ببيانات اختبار.
2. Shadow operation مع موظفي المكتب دون بيع عام كامل.
3. حجوزات حقيقية مدعوة.
4. فتح عام محدود بسقف حجوزات.
5. Stabilization ثم توسع تدريجي.

## بوابات الدخول

عقد المكتب، وثائق نافذة، مخطط مقاعد صحيح، دعم وتواصل، سياسات وأسعار، نسخ ومراقبة، نجاح AT-001..AT-035، تدريب الموظفين، وتمرين Offline/Restore.

## معايير التوسع

- صفر double booking أو فقد مال.
- لا حادث أمني حرج مفتوح.
- 14 يومًا متتاليًا مستقرة.
- حد أدنى إرشادي 30 رحلة مكتملة و300 تذكرة راكب، ويمكن تعديله بعد واقع السوق.
- تسويات ومصالحة مغلقة.
- التزام المكتب بالمخزون الموحد واستجابة الدعم.
