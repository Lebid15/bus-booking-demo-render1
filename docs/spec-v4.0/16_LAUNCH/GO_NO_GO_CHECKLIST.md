# قائمة Go / No-Go

## Go

- جميع P0 ناجحة.
- لا ثغرة عالية غير مقبولة.
- Restore drill ناجح.
- Office readiness وتدريب.
- الأسعار والسياسات والعمولة معتمدة.
- الدعم والمراقبة On-call.
- النسخة القانونية المنشورة والموافقات تعمل.

## No-Go

- احتمال بيع مزدوج.
- Ledger غير متوازن.
- Webhook غير idempotent.
- عزل مكتب غير مثبت.
- QR لا يمكن التحقق منه يدويًا.
- لا خطة للرحلات القائمة عند الإيقاف.
