# سياسة مخالفات وتعليق المكاتب

> مسودة تشغيلية كاملة للإصدار 1.0. تُراجع قانونيًا وفق القوانين السورية والجهة القضائية المختصة قبل الإطلاق التجاري. لا يجوز استخدام المراجعة القانونية لتغيير الحقوق التشغيلية المتفق عليها دون Change Request.


## مستويات المخالفة

- بسيطة: تأخر تحديث أو خطأ غير متكرر دون ضرر جوهري.
- متوسطة: تكرار خطأ، عدم رد، أو إخلال بحقوق قابل للتصحيح.
- جسيمة: بيع مزدوج، رفض تذكرة صحيحة، بيانات مضللة، تحايل عمولة.
- حرجة: تهديد سلامة، احتيال، تسريب بيانات، أو تزوير دفع/مستندات.

## الإجراءات

تنبيه، خطة تصحيح، تقييد وظيفة، منع حجوزات جديدة، Wind-down، تعليق، إنهاء وأرشفة. تحدد المدة والسبب والمراجعة.

## حماية الحجوزات القائمة

لا يؤدي التعليق إلى حذف الرحلات أو ترك الركاب دون معالجة. يقرر لكل رحلة: تنفيذ تحت رقابة، نقل بموافقة، أو إلغاء واسترداد.

## المستحقات

تجمد المبالغ المرتبطة بالمخاطر فقط، ويحتفظ باحتياطي معقول. لا تستخدم العقوبة لمصادرة أموال غير متنازع عليها دون أساس عقدي/قانوني.

## الاستئناف

يحق للمكتب الاطلاع على سبب قابل للمشاركة وتقديم اعتراض وأدلة. يمكن إبقاء الإجراء الوقائي أثناء المراجعة للمخاطر الحرجة.


## الإصدارات والموافقة

تنشر الوثيقة برقم إصدار وتاريخ نفاذ ولغة. تحفظ المنصة الإصدار الذي وافق عليه كل مستخدم أو مكتب، ولا تطبق التعديلات الجوهرية بأثر رجعي على حجز قائم.
