# ماذا لك وماذا عليك؟

## للزبون

### لك
- سعر وسياسة واضحان قبل الحجز.
- مقعد محفوظ وفق حالة الحجز والمهلة.
- تذكرة وPNR وإدارة ذاتية.
- استرداد/بديل عند إلغاء المكتب أو إخلاله وفق السياسة.
- دعم عاجل عند رفض الصعود أو مشكلة رحلة وشيكة.
- حماية بياناتك وعدم استخدامها خارج الغرض دون أساس.

### عليك
- إدخال بيانات صحيحة.
- الدفع ضمن المهلة والحضور قبل الإغلاق.
- عدم مشاركة QR أو إساءة حجز المقاعد.
- الالتزام بقواعد الأمتعة والسلوك والسلامة.

## للمكتب

### لك
- إدارة الرحلات والأسعار والسياسات ضمن حدود المنصة.
- مخزون موحد وتقارير وتسويات واعتراضات موثقة.
- حماية بياناته وعزله عن المكاتب الأخرى.
- استئناف المخالفات وتلقي سبب القرار.

### عليك
- تسجيل جميع الحجوزات في النظام.
- تنفيذ الرحلات المنشورة واحترام التذاكر.
- تأكيد الدفع بصدق والرد على الحالات العاجلة.
- حماية بيانات الركاب ودفع العمولات والمستحقات.
