# قوالب إشعارات عربية

## إنشاء حجز بانتظار الدفع

**العنوان:** تم إنشاء حجزك {pnr}
**النص:** حُفظت مقاعدك في رحلة {origin} إلى {destination} بتاريخ {departure_local}. أكمل الدفع قبل {payment_deadline_local}. حالة الحجز: {booking_status}. حالة الدفع: {payment_status}.

## تغيير جوهري

**العنوان:** مطلوب مراجعة تغيير في رحلتك {pnr}
**النص:** تغير {changed_fields}. لم نعتبر التغيير مقبولًا تلقائيًا. افتح الحجز لاختيار القبول أو البديل أو الاسترداد.

## إلغاء رحلة

**العنوان:** أُلغيت رحلة الحجز {pnr}
**النص:** ألغى المكتب الرحلة. يحق لك اختيار الاسترداد الكامل أو بديل متاح. افتح الحجز خلال الرابط الآمن.

## فشل قناة الإرسال

لا يرسل للمستخدم عبر القناة الفاشلة؛ يسجل داخل النظام ويعاد المحاولة. صفحة الحجز تظل مصدر الحقيقة.

## قواعد القالب

- لا تتضمن الرسالة QR كاملًا أو بيانات هوية حساسة.
- الرابط قصير العمر أو رمز إدارة منفصل.
- تظهر الأوقات بالتوقيت المحلي واسم المنطقة عند الالتباس.
- لا تستخدم لغة تسويقية في الرسائل الحرجة.
