# SLA وإجراءات الدعم

| الأولوية | مثال | الاستلام | أول إجراء |
|---|---|---|---|
| P0 | سلامة/اختراق/تعطل واسع | فوري | قائد حادث وعزل |
| P1 | راكب في الكراج/رفض صعود | خلال 5 دقائق أثناء التغطية | تجميد No-show وإعادة البيع والاتصال بالمكتب |
| P2 | تغيير جوهري قبل الرحلة | خلال 30 دقيقة | تأكيد الخيارات والموعد |
| P3 | استرداد/نزاع | يوم عمل | جمع الأدلة وتحديد المالك |
| P4 | استفسار | يومي | إجابة أو توجيه |

## Playbook رفض الصعود

1. افتح P1 واربط الحجز والراكب.
2. تحقق من الحالة والدفع والتذكرة والإصدار الأخير.
3. امنع No-show وإعادة البيع.
4. اتصل بالمكتب ومسؤول الرحلة.
5. نفذ أحد الحلول قبل الانطلاق قدر الإمكان.
6. وثق النتيجة والمسؤولية والتعويض.
