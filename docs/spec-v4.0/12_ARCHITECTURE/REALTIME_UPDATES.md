# التحديث اللحظي الانتقائي

## الاستخدامات

- خريطة المقاعد للحجز والمكتب.
- حالة الدفع اليدوي/الإلكتروني.
- صندوق الحالات العاجلة.
- حالة الرحلة والصعود.

## التصميم

يستخدم WebSocket أو SSE مع Authorization عند الاشتراك، ورسائل versioned لا تحتوي PII زائد. عند فقد الاتصال يعيد العميل fetch للمرجع ويستأنف من revision؛ لا يعتمد على الرسائل كحقيقة نهائية.
