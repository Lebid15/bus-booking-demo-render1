# سجل قرارات المعمارية

| ADR | القرار | السبب | البديل المرفوض |
|---|---|---|---|
| ADR-001 | Modular Monolith | معاملات قوية وفريق صغير وتشغيل أبسط | Microservices مبكرًا |
| ADR-002 | PostgreSQL مصدر المقاعد | قيود ومعاملات وقفل | Redis كمصدر نهائي |
| ADR-003 | Outbox | اتساق بين Commit والآثار الخارجية | نشر مباشر داخل المعاملة |
| ADR-004 | API-first OpenAPI | عقد موحد للواجهات والاختبارات | واجهات تنمو بلا عقد |
| ADR-005 | حساب زبون اختياري | تقليل الاحتكاك | تسجيل إلزامي |
| ADR-006 | Policies versioned + Snapshot | عدم الرجعية | قراءة الإعداد الحالي |
| ADR-007 | Web responsive first | ملاءمة السوق والكلفة | تطبيق أصلي أولًا |
