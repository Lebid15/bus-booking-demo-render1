# معايير القبول لكل Epic

> لا يُغلق Epic إلا عند نجاح جميع معايير P0 وP1 المحددة له، وربط كل معيار بدليل بناء واختبار محفوظ.

## Definition of Done المشترك

- API وDatabase migration واختبارات ووثائق محدثة.
- عزل وصلاحيات خادمية، وليس إخفاء أزرار فقط.
- Audit للعمليات الحساسة وOutbox للآثار الخارجية.
- لا أخطاء High/Critical مفتوحة في نطاق الـEpic.
- نجاح CI على PostgreSQL حقيقي.
- مراجعة RTL/Responsive للشاشات المتأثرة.
