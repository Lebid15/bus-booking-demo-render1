# تعريف الجاهزية والإنجاز

## Definition of Ready

- المتطلب مرتبط بـEpic ومعيار قبول.
- الجداول والحالات وAPI والأخطاء والصلاحيات محددة.
- التصميم يشمل loading/empty/error/permission states.
- الأثر المالي والخصوصي محدد.
- لا يوجد قرار مالك مفتوح يمنع التنفيذ.

## Definition of Done

- Code review واختبارات مناسبة ناجحة.
- Migration آمنة ومراجعة.
- OpenAPI محدث ولا كسر غير موثق.
- Audit/metrics/logging مطبقة.
- UI RTL/mobile/accessibility مراجعة.
- وثائق التشغيل والدعم محدثة.
- معايير القبول وأدلتها محفوظة.
- لا P0/P1 defect مفتوح.
