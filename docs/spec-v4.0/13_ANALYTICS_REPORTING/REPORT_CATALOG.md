# كتالوج التقارير

## المكتب

- رحلات حسب التاريخ والمسار والحالة.
- المبيعات حسب الرحلة/الفرع/الموظف/القناة.
- قائمة الركاب والصعود وNo-show.
- المدفوعات والتحويلات والفروقات.
- العمولات والتسويات.
- الوثائق المنتهية وSLA الدعم.

## المنصة

- نمو المكاتب والرحلات والحجوزات.
- التحويل والإيراد والعمولة.
- أداء الخطوط والمكاتب.
- إلغاء وتأخير وDenied Boarding.
- مخاطر ودفع واستردادات وتسويات.
- جودة المزودين وSLO.

كل تقرير يحدد timezone والعملة والفلاتر وتعريف العمود وتاريخ البيانات وحالة التصدير.
