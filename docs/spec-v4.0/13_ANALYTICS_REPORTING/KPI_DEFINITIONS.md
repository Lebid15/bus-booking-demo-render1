# تعريف مؤشرات الأداء

| KPI | التعريف |
|---|---|
| Search-to-trip-view | جلسات فتحت رحلة ÷ جلسات بحث صحيحة |
| Trip-view-to-hold | جلسات أنشأت Hold ÷ جلسات فتحت رحلة |
| Hold-to-booking | حجوزات منشأة ÷ Holds صالحة |
| Booking-to-paid | حجوزات أصبحت مدفوعة ÷ حجوزات تتطلب دفعًا |
| Seat utilization | ركاب مؤكدون/صاعدون ÷ المقاعد القابلة للبيع حسب السياق |
| Office inventory compliance | المبيعات المسجلة في النظام ÷ المبيعات الفعلية التي يقرها المكتب |
| Refund aging | الزمن من الطلب المؤهل إلى الإرسال/الإكمال |
| P1 SLA compliance | حالات P1 المستلمة ضمن SLA ÷ الكل |
| Reconciliation break rate | عمليات غير متطابقة ÷ العمليات المالية |
| Double-booking rate | يجب أن يساوي صفرًا |

تثبت التعاريف ولا تغير لرفع الرقم تاريخيًا. أي تعديل ينشئ Version ويعاد حساب المقارنة بوضوح.
