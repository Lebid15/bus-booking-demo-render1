# الأطراف والمسؤوليات

| الطرف | مسؤولياته | ما لا يملكه |
|---|---|---|
| الزبون/الراكب | صحة البيانات، الالتزام بالموعد والسياسة، إبراز الهوية عند الحاجة | تعديل السعر أو السياسة أو حالة الدفع بنفسه |
| صاحب المكتب | صحة الرحلات والأسعار والمخزون والموظفين والرد على الحالات العاجلة | تجاوز حدود المنصة أو الاطلاع على مكتب آخر |
| موظف الحجز | إنشاء وتعديل حجوزات ضمن الصلاحية وتسجيل الدفع المسموح | اعتماد استرداد مرتفع إن كان منشئه |
| موظف الصعود | التحقق والصعود وحالات الحضور | تعديل السعر أو التسوية |
| الناقل | المركبة والسائق والسلامة والتنفيذ الفعلي | تغيير حقوق الزبون بأثر رجعي |
| دعم المنصة | إدارة الحالات والتصعيد وحفظ الأدلة | تعديل القيود المالية بلا صلاحية واعتماد |
| مالية المنصة | المصالحة والتسويات والاستردادات المعتمدة | تغيير بيانات الرحلة التشغيلية |
| صاحب المنصة | السياسات العامة، التفعيل، العقود، حدود الإعدادات والرقابة | حذف التاريخ المالي أو سجل التدقيق |
