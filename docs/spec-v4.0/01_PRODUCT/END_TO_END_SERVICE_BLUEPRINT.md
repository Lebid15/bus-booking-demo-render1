# مخطط الخدمة من البداية إلى النهاية

## 1. انضمام المكتب

طلب انضمام ← جمع المستندات ← مراجعة وتحقق ← توقيع الشروط ← إنشاء المكتب والفروع ← إنشاء الموظفين ← تحديد ملف العمولة والاشتراك ← تفعيل محدود ← تفعيل كامل.

## 2. تجهيز التشغيل

إضافة الناقل والبولمان والسائقين ← إنشاء مخطط المقاعد والمجاورات ← تعريف نقاط الصعود والنزول ← إنشاء المسار ← إنشاء الرحلة والسعر والسياسة وطرق الدفع ← فحص الجاهزية ← نشر الرحلة.

## 3. الحجز العام

بحث ← نتائج وفلترة ← تفاصيل الرحلة ← اختيار المقاعد ← بيانات الركاب ← مراجعة السعر والسياسة ← Seat Hold ← اختيار الدفع ← إنشاء الحجز وPNR ← الدفع/التحقق ← إصدار التذكرة.

## 4. الحجز من المكتب

اختيار الرحلة من خريطة مباشرة ← المقعد ← بيانات الراكب أو استرجاعه ← طريقة الدفع ← تأكيد وطباعة/إرسال.

## 5. ما قبل الرحلة

تذكير وتعليمات ← تحديثات جوهرية تحتاج قبولًا ← تغيير مركبة/مقعد مع إعادة تحقق ← دعم واستفسارات ← إغلاق الحجز عند الموعد.

## 6. الصعود

فتح الصعود ← QR أو تحقق يدوي ← حالة مستقلة لكل راكب ← معالجة QR مكرر أو قائمة دون اتصال ← نداء أخير ← إغلاق الصعود ← منع No-show عند وجود نزاع أو إخلال مكتب.

## 7. تنفيذ وإغلاق الرحلة

تسجيل الانطلاق الفعلي ← قفل Manifest ← الوصول ← إغلاق تشغيلي ← معالجة الحالات المفتوحة ← إغلاق مالي للرحلة مع تجميد المتنازع عليه فقط.

## 8. المال

إثبات العمولة ← مصالحة الدفع ← استردادات/اعتراضات ← إنشاء دورة تسوية ← مراجعة واعتماد ← تحويل/تحصيل ← إغلاق مع قيود دفتر متوازنة.

## 9. ما بعد الرحلة

تحميل التذكرة/الإيصال ← تقييم اختياري ← دعم المفقودات والشكاوى ← تقارير الأداء ← مؤشرات المخاطر والتحسين.
