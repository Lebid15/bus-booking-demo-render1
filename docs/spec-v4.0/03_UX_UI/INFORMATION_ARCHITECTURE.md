# معمارية المعلومات

## الموقع العام

- الرئيسية والبحث.
- نتائج الرحلات.
- تفاصيل الرحلة والسياسة والخدمات.
- اختيار المقاعد.
- بيانات الركاب والمرافقين.
- مراجعة الحجز والدفع.
- نتيجة الحجز والتذكرة.
- إدارة الحجز/PNR.
- تسجيل الدخول/إنشاء الحساب الاختياري.
- ملفي وحجوزاتي.
- مركز المساعدة والسياسات.

## لوحة المكتب

- الرئيسية التشغيلية.
- الرحلات وخريطة المقاعد الحية.
- الحجوزات والمدفوعات.
- الصعود وقوائم الركاب.
- البولمانات ومخططات المقاعد والسائقون.
- المسارات والنقاط والأسعار والسياسات.
- الفروع والموظفون والصلاحيات.
- الدعم والحالات العاجلة.
- التقارير والعمولات والتسويات والاشتراك.
- إعدادات المكتب وسجل التدقيق المتاح له.

## لوحة المنصة

- نظرة عامة ومراقبة النظام.
- المكاتب والتحقق والعقود.
- المواقع والمسارات والناقلون.
- الاشتراكات والعمولات والتسويات.
- الحجوزات والمدفوعات والاستردادات والنزاعات.
- الدعم والحوادث والمخالفات والمخاطر.
- السياسات والإعدادات والإشعارات.
- الأدوار والتدقيق والخصوصية والاحتفاظ.
- التقارير والتحليلات والإطلاق التدريجي.

## قواعد التنقل

- يظهر للمستخدم الإجراء الأكثر شيوعًا أولًا ولا يُدفن داخل قائمة “المزيد”.
- كل كيان تشغيلي يفتح في صفحة أو Drawer واضحة، ولا تعتمد العمليات الحساسة على Tooltip فقط.
- العودة من التفاصيل تحفظ البحث والفلترة والصفحة السابقة.
- الروابط العميقة آمنة وتتحقق من الصلاحية خادميًا.
- لا تعرض الواجهة أي مبلغ أو بيانات شخصية قبل تحقق الصلاحية.
