# الاستجابة وإمكانية الوصول

## نقاط العرض

- هاتف صغير: 360px فأعلى.
- هاتف/لوحي: 480–767px.
- لوحي: 768–1023px.
- سطح مكتب: 1024px فأعلى.

## قواعد الهاتف

- البحث في بطاقة واحدة متدرجة الخطوات.
- نتائج الرحلات بطاقة عمودية، لا جدول أفقي.
- خريطة المقاعد يمكن تكبيرها دون فقد السياق، وتبقى أزرار المتابعة ثابتة.
- جداول التقارير تتحول إلى بطاقات أو تمرير أفقي مع أول عمود ثابت.
- الإجراءات الخطرة لا توضع قرب زر الرجوع أو الحفظ.

## الوصول

- كل مقعد زر له اسم: “المقعد 12، متاح، نافذة”.
- التنقل بلوحة المفاتيح داخل الخريطة بأسهم واتجاه منطقي.
- Focus مرئي لا يُزال.
- الأخطاء مرتبطة بالحقول بواسطة aria-describedby.
- الحد الأدنى للأهداف اللمسية 44×44px.
- التباين AA، مع اختبار Dark Mode إن أضيف لاحقًا.
- لا تستخدم CAPTCHA بصريًا فقط؛ يوفر بديلًا مناسبًا.
- المستندات المهمة قابلة للقراءة والطباعة وليست صورًا فقط.
