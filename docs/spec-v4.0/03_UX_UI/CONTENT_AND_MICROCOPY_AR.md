# المحتوى والنصوص العربية

## مبادئ

- لا تستخدم “فشل” دون توضيح ما بقي محفوظًا وما يمكن فعله.
- لا تقل “تم الحجز” إذا كان غير مدفوع؛ استخدم “تم إنشاء الحجز وبانتظار الدفع”.
- لا تكشف سبب منع المقعد المرتبط بجنس راكب آخر؛ استخدم “المقعد غير متاح وفق سياسة توزيع المقاعد”.
- اذكر الوقت المطلق مع العد التنازلي: “ينتهي الحجز الساعة 6:30 مساءً (بعد 18 دقيقة)”.
- افصل حالة الحجز عن حالة الدفع.

## نصوص معيارية

| السياق | النص |
|---|---|
| Seat Hold | تم حفظ المقاعد مؤقتًا. أكمل البيانات قبل انتهاء المهلة. |
| انتهاء Hold | انتهت مهلة حفظ المقاعد. أعد اختيار المقاعد المتاحة. |
| نجاح غير مدفوع | تم إنشاء الحجز. يبقى المقعد محفوظًا حتى الموعد الموضح لإكمال الدفع. |
| إثبات تحويل | وصل إثبات التحويل وهو قيد المراجعة. لا تعتبر الدفعة مؤكدة حتى تتغير الحالة إلى «مدفوع». |
| فشل بريد | حجزك ناجح، لكن تعذر إرسال البريد. احتفظ برقم الحجز أو نزّل التذكرة من هذه الصفحة. |
| QR مكرر | تم استخدام هذه التذكرة للصعود مسبقًا. راجع موظف الصعود إن كنت تعتقد أن هناك خطأ. |
| تغيير جوهري | تغيرت تفاصيل الرحلة. راجع التغيير واختر قبول الرحلة أو البديل أو الاسترداد. |
| إلغاء | سيؤدي هذا الإجراء إلى تحرير المقعد وبدء الاسترداد الموضح أدناه. |
| لا نتائج | لا توجد رحلات مطابقة. جرّب تاريخًا آخر أو نقطة قريبة. |
| خطأ عام | تعذر إكمال العملية. لم يتم خصم مبلغ جديد. رقم المتابعة: {request_id}. |
