## متطلبات مشتركة

- كل Mutation يعرض Loading ويمنع النقر المكرر ويستخدم Idempotency عند الحساسية.
- لا تُعرض أزرار لا يملك المستخدم صلاحيتها؛ مع بقاء التحقق خادميًا إلزاميًا.
- كل شاشة لها حالات Loading وEmpty وError وPartial/Degraded.
- يظهر رقم الطلب عند الخطأ القابل للدعم.
