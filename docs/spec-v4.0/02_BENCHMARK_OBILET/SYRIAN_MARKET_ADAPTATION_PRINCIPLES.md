# مبادئ تكييف النموذج للسوق السوري

1. **الدفع الواقعي أولًا:** لا يشترط الحساب البنكي أو البطاقة؛ تُفعل القنوات وفق المكتب والسوق.
2. **الثقة بالتدرج:** إظهار المكتب والناقل والسياسة والسعر ومرجع الحجز بوضوح، وعدم إخفاء حالة الدفع.
3. **الويب قبل التطبيق:** واجهة سريعة على الهاتف والمتصفح، مع أقل حجم ممكن للصفحات.
4. **العمل عند ضعف الاتصال:** لا بيع Offline، لكن صعود طوارئ من Manifest مشفر محدود المدة.
5. **المكتب يستفيد فورًا:** الحجز الداخلي والخارجي في خريطة واحدة، لا إدخال مزدوج.
6. **السياسات مرنة لكن محكومة:** المكتب يختار ضمن حدود المنصة، ولا يغير حقوق حجز قائم.
7. **اللغة محلية وواضحة:** مصطلحات بسيطة، أرقام عربية/لاتينية مدعومة، ورسائل لا تفترض خبرة تقنية.
8. **التوسع بعد الإثبات:** يبدأ بممر ومكاتب محدودة، ثم تتوسع المدن والميزات تدريجيًا.
