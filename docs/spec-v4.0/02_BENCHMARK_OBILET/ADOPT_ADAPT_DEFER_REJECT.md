# قرارات المقارنة

| الميزة | القرار | التطبيق في مشروعنا |
| --- | --- | --- |
| بحث متعدد الشركات | ADOPT | عرض مقارنة واضح ضمن المكاتب المفعلة |
| شراء إلكتروني فوري فقط | ADAPT | دفع مرن: مكتب/تحويل/إلكتروني |
| PNR وإدارة دون حساب | ADOPT | PNR + رمز إدارة آمن |
| قاعدة جنس المقاعد | ADOPT/HARDEN | تحقق ذري ومجاورة معرفة بالمخطط |
| سياسات إلغاء حسب الناقل | ADOPT | قوالب وحدود منصة وSnapshot |
| تذكرة مفتوحة | DEFER | قسيمة مستقبلية بعد نضج المالية |
| تأمين/حماية إلغاء | DEFER | منتج إضافي لاحق |
| دعم أولوية مدفوع | REJECT PH1 | الدعم العاجل حق تشغيلي لا حزمة مدفوعة |
| تطبيق أصلي | DEFER | ويب متجاوب أولًا |
| طبقة وكلاء B2B | DEFER | لوحة المكتب أولًا ثم API خارجي |
| إرسال SMS وبريد | ADAPT | البريد أولًا وSMS عند توفر مزود؛ صفحة الحجز مصدر الحقيقة |
| فوترة إلكترونية تركية | ADAPT | إيصال/تذكرة بحسب المتطلبات السورية النهائية |
